Information on what I have done

In this example I created a graphical bar chart that reflects the information in the data table below.

This example was created as an AngularJS single page web application and it consists of a main controller "nccTest" which contains the base data used for the bar chart. I also created a separate directive "theInputTable" which contains all of the base data and functional logic related to the data table found below the bar chart.

In this example I used:

  • AngularJS - a powerful MVC framework used to create scalable maintainable web applications
  • n3-charts - an AngularJS library built on top of the popular D3.js a popular library for manipulating documents based on data.
  • Smart Table - an Angularjs module used to easily display data in a table with a set of built in features such filtering,sorting, etc in a declarative way.
These tools allowed me to quickly create a demo with all of the requirements in the document I received, here within the 2 hour time limit.

However with extra time I would have included:

  • Input field type checking
  • Duplicate checking - disallow duplicate data to be entered
  • Fix bug with bar alignment when there is only a single bar being displayed in the bar chart
  • Included styling to highlight a table column header when filtering by header
  • Appropriate CSS classes (no inline styling)
  • Fully responsive styling on all page elements
  • Fully commented / tested code
